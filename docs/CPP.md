# Verwendung aus C++

## Adapter

```cpp
LayerShellAdapter adapter;
```

---

## Konfiguration

```cpp
LayerShellConfig config;
```

---

## Initialisierung

```cpp
adapter.initialize(
    widget,

```

## Eigenschaften

Unterstützt werden aktuell:

- Layer
- Screen
- Anchors
- Margins
- Exclusive Zone
- Desired Size
